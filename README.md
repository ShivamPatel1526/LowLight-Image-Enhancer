# 🌙 Low-Light Image Enhancer

> A Python project to enhance low-light images using histogram equalization and OpenCV.

This tool improves dark, underexposed images by adjusting brightness intelligently while preserving color accuracy. It uses the HSV color space to perform histogram equalization and displays a **side-by-side comparison** of the input and output images with labeled overlays.

---

## 🖥️ Built With

- 🐍 Python 3.x
- 📦 OpenCV
- 🔢 NumPy

---

## 📸 Features

- 🌈 Brightness enhancement via HSV color model
- 🪄 Preserves natural color tones
- 🪟 Input and Output shown side-by-side
- 🏷️ Clear, bold labels above both images
- 💾 Automatically saves enhanced output
- 📐 Fits display size with smooth resizing
