# 🖼️ Low-Light Image Enhancer in Python

A Python tool that improves low-light images using **color histogram equalization** with OpenCV.

---

## 👨‍💻 Features

- Enhances dark or underexposed images  
- Uses HSV color space to preserve color accuracy  
- Input and output images shown side-by-side  
- Clear labels above each image for easy comparison  
- Output saved automatically to local storage  

---

## 📁 File Structure

```
LowLight-Image-Enhancer/
├── enhancer.py           # Main script  
├── requirements.txt      # Dependencies  
├── sample.jpg            # (Optional) Input image  
├── output.jpg            # (Generated) Enhanced image  
```

---

## 🚀 How to Run

### 🛠 Prerequisites  
- Python 3.x  
- OpenCV (`opencv-python`)  
- NumPy  

### ⚙ Install Dependencies  
```bash
pip install -r requirements.txt
```

### ▶ Run the Script  
```bash
python enhancer.py
```

---

## 📸 Sample Output

- Displays a side-by-side window:  
  - Input Image (original)  
  - Enhanced Image (brighter and clearer)  
- Saves the enhanced version as: `output.jpg`
